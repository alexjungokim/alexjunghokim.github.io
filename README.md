# alexjunghokim.github.io

Personal website and blog built with [al-folio](https://github.com/alshedivat/al-folio).

**Live at:** [alexjunghokim.github.io](https://alexjunghokim.github.io)

## Local Development

```bash
bundle install
bundle exec jekyll serve
# Site available at http://localhost:4000
```

## Structure

- `_posts/` — Blog posts (fundamentals, paper reviews, system design, personal)
- `_projects/` — Project write-ups
- `_bibliography/papers.bib` — Publications
- `_data/cv.yml` — CV data
- `_pages/` — Site pages (about, blog, publications, projects, CV)
